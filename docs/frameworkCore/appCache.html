appCache

Description
Description | Methods (details)

This class is responsible for caching with memcached.

To enable caching, set "memcache_host" config setting to appropriate value. If "memcache_host" is not set, caching will be disabled.

  • author: Paulina Budzon <paulina.budzon@gmail.com>
  • uses: Memcached - to cache the data.
