		static sthrowError (line 229)
	
 
	
	

Throws given error and ends the request. If the appropriate error template exists it will be shown.


To use the template name the appropriate file in templates/ as "error_numberofError.php", for example "error_404.php".  If a debug mode is on ("debug" option in appConfig), then it will also include $message in the output.
