		render (line 79)
	
 
	
	

Main function called to render the views.


This returns false if views are disabled ($_disabled).  It returns ONLY ajax or ONLY "normal" templates:  if $_isAjax is set to true and there are some ajax templates added - the call will be returned as json encoded string.  In other case "normal" html will be returned to the user.

	
    
				
  • return: Returns false if views are disabled.
